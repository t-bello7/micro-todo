# Micro-Todo-list

>A minimalist Todo list app that's everywhere you are. "To-do list" is a tool that helps to organize your day. It simply lists the things that you need to do and allows you to mark them as complete. 

## Built With

- HTML5
- CSS3
- Javascript
- markdowns
